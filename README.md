## How to run the GUI

Use these steps on Windows PowerShell to build and launch the GUI:

```powershell
cd src
javac *.java
javac -d ..\bin *.java
cd ..\bin
java Main
```

In the GUI:
- Click “IPL” and select the `load_file.txt` that was generated by assembling the `.asm` program and Press **Run**.
- In the “Console Input” box, paste a full input line and press **Enter** once.
- Make sure Target Input is typed followed by 20 numbers.
- For example, 
4 9 9 9 9 9 9 9 9 9 9 3 5 9 9 9 9 9 9 9 9

Here target is 4 and the answer would be 3.
- The program’s output appears as a single continuous number in the Printer area.

To assemble the fixed-20 program first (so `load_file.txt` exists):

```powershell
cd src
javac *.java
java Assembler ..\closest_20.asm
javac -d ..\bin *.java
cd ..\bin
java Main
```
